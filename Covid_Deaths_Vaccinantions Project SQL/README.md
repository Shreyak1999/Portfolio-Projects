Covid Deaths and Vaccinations Project

This project analyzes COVID-19 deaths and vaccination trends using data sourced from public datasets. The analysis is done using SQL and focuses on exploring global patterns, comparing vaccination rollouts, and examining relationships between case numbers, death rates, and vaccination progress.

📁 Project Structure

    Main SQL script containing queries to clean, transform, and analyze the COVID-19 deaths and vaccinations datasets.

📊 Key Features

    Analysis of COVID-19 death rates by country and continent.

    Identification of trends in vaccinations over time.

    Calculations of metrics like death percentage, vaccination percentage, and population impact.

    Use of temporary tables and CTEs for modular, readable SQL.

    Joining datasets for enriched insights (e.g., correlating vaccinations with death trends).

🛠 Technologies Used

    SQL: Primary tool for querying and transforming data.

    Relational Database: Assumes compatibility with systems like MySQL, PostgreSQL, or SQL Server.

🔍 Example Analyses

    Total deaths vs. population by continent.

    Countries with the highest death-to-population ratio.

    Daily global vaccinations.

    Rolling counts of vaccinated people.

📌 How to Use

    Load the relevant COVID-19 datasets (CovidDeaths.csv and CovidVaccinations.csv) into your database.

    Run the queries provided in the .sql file to perform the analysis.

    Modify queries as needed to suit your specific database engine or research questions.
