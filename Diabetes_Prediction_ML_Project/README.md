# Diabetes Prediction ML Portfolio Project

This project demonstrates the use of machine learning algorithms to predict the likelihood of diabetes in individuals based on a set of medical parameters. It is part of a personal data science and ML portfolio.

📁 Project Structure

Diabetes_Prediction_ML_Portfolio_Project.ipynb – Main notebook containing the data preprocessing, exploratory data analysis, model building, and evaluation.

📊 Dataset

The dataset used in this project is the Pima Indians Diabetes Database, which includes various health-related variables:

    Pregnancies
    Glucose
    Blood Pressure
    Skin Thickness
    Insulin
    BMI
    Diabetes Pedigree Function
    Age
    Outcome (0: No diabetes, 1: Diabetes)

🔍 Project Steps

    Data Loading: Importing the dataset using pandas.
    Data Cleaning: Handling missing values and outliers.
    Exploratory Data Analysis (EDA): Visualizing distributions and relationships using matplotlib and seaborn.
    Feature Engineering: Scaling and transforming features.

Modeling:

    Logistic Regression
    Decision Tree
    Random Forest
    Support Vector Machine (SVM)
    
Evaluation: Using metrics like accuracy, precision, recall, F1-score, and ROC-AUC.

Model Selection: Comparing models to identify the best performing one.

Conclusion and Insights: Summary of findings and future scope.
