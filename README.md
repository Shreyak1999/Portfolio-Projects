## 📂 Project List

### 1. 📊 Causal Impact of Women’s Education on Domestic Violence in Odisha
- **Data:** NFHS-5 Odisha
- **Tools:** Python, DoWhy, EconML, Pandas
- **Focus:** Causal inference, policy analytics, public health
- **Summary:** Estimated the causal effect of women’s secondary education on their likelihood of experiencing domestic violence using NFHS-5 survey data and modern causal inference techniques.

## 📌 Technologies Used

- **Languages:** Python, SQL  
- **ML/Stats:** scikit-learn, XGBoost, DoWhy, EconML  
- **Data:** NFHS-5, Census, Open Government Data, Other Data from Kaggle
- **Visualization:** Power BI, Tableau, Matplotlib  
- **Deployment:** Streamlit, Flask
