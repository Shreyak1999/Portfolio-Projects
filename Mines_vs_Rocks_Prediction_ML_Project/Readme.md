Project Overview
This project focuses on building a machine learning model to classify sonar signals as either Mines or Rocks. It uses data from sonar returns to predict the type of object detected under the surface. This project showcases supervised classification modeling as part of a machine learning portfolio.

Dataset
Source: UCI Machine Learning Repository — Sonar Dataset

Features: 60 numeric attributes representing energy within a specific frequency band

Target:

M — Mine
R — Rock

Project Workflow

  Load dataset
  Understand data dimensions and structure
  Exploratory Data Analysis (EDA)
Data Preprocessing
  Label encoding of target variable
  Train-test split
  Data normalization
Model Building
  Logistic Regression model
  Model training on training data
Model Evaluation
  Accuracy score
  Confusion matrix
Classification report
  Prediction
  Model testing with sample inputs
Result interpretation
